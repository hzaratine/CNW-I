Informática para Internet

ETEC

Este site foi criado para apresentar o curso técnico de Informática para Internet oferecido pela ETEC Maria Cristina Medeiros. É um curso voltado para quem gosta de tecnologia e quer entender como funcionam sites, sistemas e aplicações que usamos todos os dias.

Durante o curso, o aluno aprende desde os conceitos básicos de programação até a criação de páginas completas, trabalhando tanto o visual de um site quanto a parte lógica(back-end) que faz tudo funcionar por trás.

Além de aprender linguagens importantes como HTML, CSS e JavaScript, o curso também aborda banco de dados, redes e desenvolvimento de sistemas, o que ajuda bastante na preparação para o mercado de trabalho.

No final da formação, o estudante consegue criar projetos reais, como sites institucionais, sistemas de cadastro, páginas responsivas e aplicações web mais completas.

Sobre o Curso

O curso técnico de Informática para Internet é ideal para quem quer aprender a desenvolver sites, sistemas e aplicações que funcionam através da web. Ele mistura teoria e prática, permitindo que o aluno entenda como as tecnologias são usadas no dia a dia.

No começo do curso, os alunos aprendem a base da lógica de programação e o funcionamento dos computadores. Depois disso, começam a desenvolver páginas e projetos, entendendo como funciona a estrutura de um site e como ele pode ser planejado.

Conforme o curso avança, entram matérias mais completas como banco de dados, redes e desenvolvimento de sistemas. Isso faz com que o aluno aprenda a criar projetos maiores, parecidos com os que são usados em empresas.

Além da parte técnica, o curso também desenvolve habilidades importantes como raciocínio lógico, organização, criatividade e trabalho em equipe, já que muitas atividades e projetos são feitos em grupo.

Ao final da formação, o aluno sai com uma boa base para atuar como estagiário, desenvolvedor iniciante ou até seguir para uma faculdade na área de tecnologia.

Grade Curricular

O curso é dividido em três anos. abaixo esta as matérias.

1º Ano

Programação e Algoritmos

Introdução à lógica de programação e resolução de problemas.

Fundamentos da Informática

Hardware, software e funcionamento dos computadores.

Interfaces Web I (IWI)

Criação de páginas usando HTML e CSS.

Arte Digital

Noções de design e criação visual.

Banco de Dados

Organização e armazenamento de informações.

2º Ano

Desenvolvimento de Software

Criação de sistemas utilizando programação.

Interfaces Web II

JavaScript e interatividade em páginas web.

DDM

Design digital e mídias interativas.

Redes de Computadores

Funcionamento da internet e comunicação entre computadores.

3º Ano

Programação para Internet

Desenvolvimento de sistemas web completos.

Projeto de Aplicação Web

Projetos integrados simulando situações reais.

TCC

Projeto final reunindo os principais conteúdos do curso.

Mercado de Trabalho

A área de tecnologia está em constante crescimento e profissionais que trabalham com internet são muito procurados. Hoje praticamente toda empresa precisa de um site, de um sistema interno ou de alguma solução online.

O técnico em Informática para Internet pode atuar em diversas áreas, desde a criação de páginas simples até sistemas completos para empresas, com login, cadastro e banco de dados.

  • Criação e manutenção de sites
  • Desenvolvimento de sistemas web
  • Programação front-end e back-end
  • Design e criação de interfaces
  • Suporte técnico e manutenção de sistemas
  • Trabalhos como freelancer e projetos próprios

O curso também prepara o aluno para continuar estudando na área de tecnologia e buscar oportunidades maiores, como trabalhar em empresas de desenvolvimento ou até criar seus próprios projetos e negócios digitais.
